# trucks-api-master


## Installation

### Dependencies

### For Back :

#### Install npm

```
npm install
npm run build
```

### Start server
`npm start`


## Application 
URL : http://localhost:4000/api

### For Front :

`cd front`

`npm install`

`ember s`

URL : http://localhost:4200/home
